# Style

## Overview

Formatting and style enforcement skills for documentation and code. This plugin ensures consistent markdown formatting by enforcing lint rules and project conventions.

- `/style:markdown` - Enforce markdown formatting rules when creating or editing `.md` files

## Skills

| Skill              | Description                                                              |
| ------------------ | ------------------------------------------------------------------------ |
| `style:markdown`   | Enforce markdown formatting conventions and non-auto-fixable lint rules  |

## Complete Examples

### /style:markdown

This skill takes no arguments. It is automatically invoked when creating, editing, or reviewing markdown files.

```bash
# Review a markdown file for style issues
/style:markdown
```
